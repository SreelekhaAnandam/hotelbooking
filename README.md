# Hotel Booking System

A simple web-based hotel booking application that allows users to explore hotels, search rooms, and make reservations.

## Features
- Browse available hotels and rooms
- Search hotels based on preferences
- User login and registration pages
- Booking and payment interface
- Responsive design for better user experience

## Technologies Used
- HTML
- CSS
- JavaScript
